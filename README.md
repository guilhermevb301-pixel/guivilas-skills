# Skills do Gui — Claude Code

Skills premium para criar landing pages de alta conversão com Claude Code.

---

## Skills disponíveis

### 1. `paginadevendas-guiinfo` — Página de Vendas Info
Cria landing pages completas para cursos e produtos digitais. Inclui carrossel de módulos, depoimentos 3D, bloco de preço e efeitos premium.

### 2. `paginadevendas-gui-highend` — Página High-End
Versão avançada com intake completo (5 blocos), hero com vídeo ambiente, Clash Display + Inter, animações premium e orquestração de imagens no Higgsfield/Seedance.

---

## Como instalar

1. Baixe ou clone este repositório
2. Copie a pasta da skill desejada para `~/.claude/skills/`

```bash
# Clonar o repositório
git clone https://github.com/guilhermevb301-pixel/guivilas-skills.git

# Copiar as skills para o Claude Code
cp -r guivilas-skills/paginadevendas-guiinfo ~/.claude/skills/
cp -r guivilas-skills/paginadevendas-gui-highend ~/.claude/skills/
```

3. No Claude Code, use a skill digitando:

```
/paginadevendas-guiinfo
```
ou
```
/paginadevendas-gui-highend
```

---

## Requisitos

- [Claude Code](https://claude.ai/code) instalado
- (Opcional) Conta no Higgsfield para geração de imagens dos módulos
- (Opcional) Conta no Seedance para geração do vídeo hero
